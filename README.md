# Casbin.NET SqlSugar Adapter

[![NuGet](https://img.shields.io/nuget/v/SharpFort.Casbin.Adapter.SqlSugar.svg)](https://www.nuget.org/packages/SharpFort.Casbin.Adapter.SqlSugar/)

SqlSugar adapter for [Casbin.NET](https://github.com/casbin/casbin-net). 

With this library, Casbin can load policy from databases supported by [SqlSugar](https://github.com/donet5/SqlSugar) or save policy to them. 

## Supported Databases

Basically, any database supported by SqlSugar is supported by this adapter:
- MySQL
- PostgreSQL
- SQL Server
- Oracle
- SQLite
- ClickHouse
- ...and more.
